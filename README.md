# ECOLIGHT-MAESTRO: Time & Intensity Adaptive Street Light Automation

## Overview

ECOLIGHT-MAESTRO is an embedded system project developed using the LPC2129 ARM7 microcontroller. The system automatically controls street lights based on the current time using the Real-Time Clock (RTC). It also provides a keypad-based interface for editing RTC settings and displays information on a 16×2 LCD.

---

## Features

* Automatic street light control
* RTC-based time monitoring
* LCD display for date and time
* Keypad-based RTC editing
* External interrupt for RTC menu access
* Embedded C implementation

---

## Hardware Components

* LPC2129 ARM7 Microcontroller
* 16×2 LCD Display
* 4×4 Matrix Keypad
* RTC
* LDR Sensor
* LED
* Push Button
* Power Supply

---

## Software Used

* Embedded C
* Keil uVision
* Proteus

---

## Working Principle

The system continuously monitors the current time using the RTC. During night hours, the street light is turned ON automatically, while it remains OFF during daytime. The LCD displays the current time and date, and the keypad allows the user to edit the RTC settings through a menu interface.
